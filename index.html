I got my first computer a Sinclair Spectrum ZX +2 in 1989. Soon to be replaced by an all mighty 80386. I wrote my first programs in QBasic, and later learned C and Turbo Pascal.

I studied a double degree in Mathematics and Engineering. Soon after I worked in assistant positions in the algebra and logic departments across Europe. But finally I settled as a Software Engineer.

I enjoy programming challenges, like Leet Code or Advent of Code. Also graphics and physics engines, algorithms and data structures, memory management, language and type theory and everything in between.

My favorite language is Rust and I am most exited about Web Assembly, WebGPU and Web Components.
